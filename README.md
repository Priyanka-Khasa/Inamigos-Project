# InAmigos Foundation — NGO Awareness Website

A responsive single-page website built to spread awareness about **InAmigos Foundation**, a Section 8 registered non-profit organization working across India in education, food distribution, animal welfare, women empowerment, environmental sustainability, and skill development.

## 📖 About

InAmigos Foundation was founded on September 23, 2020, and is based in Chhattisgarh. It is 80G & 12A certified, CSR-1 registered, NITI Aayog registered, and ISO 9001:2015 certified. This site was built to showcase the foundation's mission, ongoing projects, and impact — and to drive support and volunteer engagement.

## ✨ Features

- **Responsive navbar** with mobile menu toggle
- **Hero section** introducing the foundation's mission
- **About section** highlighting certifications and credibility
- **Projects section** covering all 6 core initiatives:
  - 🍲 Project Seva — Food & clothing distribution
  - 📚 Project Bachpanshala — Education & digital literacy for children
  - 🐾 Project Jeev — Animal welfare & rescue
  - 👗 Project Udaan — Women empowerment & skill development
  - 🌱 Project Prakriti — Environmental conservation
  - 💼 Project Vikas — Internship-based skill development
- **Impact section** displaying key stats (beneficiaries, states, volunteers)
- **Gallery** showcasing campaign visuals
- **Support/CTA section** with links to donate, volunteer, or contact the NGO
- **Social links** to Instagram, LinkedIn, and the official website

## 🛠️ Tech Stack

- **HTML5** — Page structure
- **CSS3** — Styling and responsive layout
- **JavaScript** — Interactivity (mobile nav toggle, etc.)
- **Font Awesome** — Icons

## 🚀 Getting Started

1. Clone the repository
   ```
   git clone https://github.com/Priyanka-Khasa/Inamigos-Project.git
   ```
2. Navigate into the project folder
   ```
   cd Inamigos-Project
   ```
3. Open `index.html` directly in your browser — no build step or dependencies required.
